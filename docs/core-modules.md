# Core Module API Reference

This document describes the internal APIs of the rNES emulator core. These are the
functions called from the platform entry point to initialize, reset, clock, and
destroy each subsystem. All information is derived from the Doxygen documentation
generated by `make docs`.

Platform porters interact with these modules in the entry point
(`src/interface/<platform>/main.c`) but should **not** call them from inside HAL
implementation files. The core is entirely platform-independent.

## System Bus (sbus)

**Header:** `src/include/sbus.h`  
**Implementation:** `src/sbus.c`

The system bus connects the CPU, PPU, APU, cartridge, and controller. It exposes a
unified 16-bit address space and is the single clocking point for the entire emulator.

### Types

```c
typedef struct sbus_t;    /* System BUS */
typedef enum   mem_op_t;  /* Memory operation: READ or WRITE */
```

### Functions

---

#### `sbus_init`

```c
int sbus_init(void);
```

Initialize the system bus internal structures.

**Returns:** `0` on success, error code otherwise.

---

#### `sbus_destroy`

```c
int sbus_destroy(void);
```

Finalize the system bus and free its resources.

**Returns:** `0` on success, error code otherwise.

---

#### `sbus_set_cartridge`

```c
int sbus_set_cartridge(cartridge_t *cartridge);
```

Attach a loaded cartridge to the system bus.

| Parameter | Direction | Description |
|---|---|---|
| `cartridge` | in | Cartridge struct returned by `cartridge_load()`. |

**Returns:** `0` on success, `-1` otherwise.

---

#### `sbus_read` / `sbus_write`

```c
uint8_t sbus_read(uint16_t address);
void    sbus_write(uint16_t address, uint8_t value);
```

Read or write one byte (8 bits) on the system bus.

| Parameter | Direction | Description |
|---|---|---|
| `address` | in | 16-bit memory address. |
| `value` | in | Data byte to write (`sbus_write` only). |

`sbus_read` returns the byte stored at `address`.

---

#### `sbus_clock`

```c
void sbus_clock(void);
```

**System bus clock.** Handles clocks from all modules and DMA transactions.

This is the central function called by the emulator thread. Each call advances
the CPU, PPU, and APU by one master clock cycle and handles any pending OAM DMA
transfer (which stalls the CPU via `cpu_suspend()` / `cpu_wakeup()`).

---

#### `sbus_register_debug_cb` / `sbus_unregister_debug_cb`

```c
int sbus_register_debug_cb(void (*debug_cb)(mem_op_t, uint16_t, uint8_t));
int sbus_unregister_debug_cb(void);
```

Register or unregister a debug callback that is invoked on every bus read/write.

> **⚠ Not thread safe.** Register before starting the emulator thread.

`sbus_register_debug_cb` returns `0` on success, `-1` if a callback is already
registered. Enabling this callback causes extreme slowdown.

## CPU — MOS 6502 (cpu650x)

**Header:** `src/cpu/include/cpu650x.h`  
**Implementation:** `src/cpu/cpu650x.c`

Full emulation of the MOS 6502 processor as used in the NES (Ricoh 2A03, which
omits the BCD mode).

### Types

```c
typedef struct cpu650x_t;        /* CPU 650x registers and state */
typedef struct cpu650x_state_t;  /* CPU state snapshot passed to callbacks */
```

### Functions

---

#### `cpu_init` / `cpu_destroy`

```c
int cpu_init(void);
int cpu_destroy(void);
```

Initialize and finalize CPU internal structures.

**Returns:** `0` on success, `-1` otherwise.

---

#### `cpu_reset`

```c
void cpu_reset(void);
```

Reset the CPU to its initial power-on state (loads the reset vector, clears
registers, sets the interrupt disable flag).

---

#### `cpu_clock`

```c
void cpu_clock(void);
```

Perform one master clock tick in the CPU. Called internally by `sbus_clock()`.
Executes one 6502 instruction cycle (instructions take 2–7 cycles).

---

#### `cpu_suspend` / `cpu_wakeup`

```c
int cpu_suspend(void);
int cpu_wakeup(void);
```

Pause and resume the CPU (used by `sbus_clock()` during OAM DMA).

**Returns:** `0` on success, `-1` otherwise.

---

#### `cpu_get_state`

```c
enum _CPU_STATE cpu_get_state(void);
```

Return the current CPU state (running, suspended, killed, etc.).

---

#### `cpu_trigger_irq` / `cpu_clear_irq`

```c
void cpu_trigger_irq(void);
void cpu_clear_irq(void);
```

Assert and deassert the maskable interrupt (IRQ) line.

---

#### `cpu_trigger_nmi` / `cpu_clear_nmi`

```c
void cpu_trigger_nmi(void);
void cpu_clear_nmi(void);
```

Assert and release the non-maskable interrupt (NMI) line. Used by the PPU to
signal VBlank.

---

#### `cpu_get_cycles`

```c
unsigned long cpu_get_cycles(void);
```

Return the total number of master clock cycles executed since reset.

**Returns:** `unsigned long` cycle count.

---

#### `cpu_register_kill_cb` / `cpu_unregister_kill_cb`

```c
int cpu_register_kill_cb(void (*kill_cb)(cpu650x_state_t));
int cpu_unregister_kill_cb(void);
```

Register a callback invoked when the CPU executes an illegal opcode and halts.
The reference entry point uses this to display the "CPU KILLED!" error overlay.

| Parameter | Description |
|---|---|
| `kill_cb` | Function receiving a `cpu650x_state_t` snapshot at the time of kill. |

> **⚠ Not thread safe.** Register before starting the emulator thread.

**Returns:** `0` on success, `-1` if a callback is already registered.

---

#### `cpu_register_debug_cb` / `cpu_unregister_debug_cb`

```c
int cpu_register_debug_cb(void (*debug_cb)(cpu650x_state_t));
int cpu_unregister_debug_cb(void);
```

Register a callback invoked after every instruction. Causes extreme slowdown.

> **⚠ Not thread safe.** Register before starting the emulator thread.

## PPU — Picture Processing Unit

**Header:** `src/include/ppu.h`  
**Implementation:** `src/ppu.c`

The PPU renders the 256×240 NES picture to the display via `gui_set_pixel()`.

### Types

```c
typedef struct ppu_t;      /* PPU state */
typedef enum   ppu_video_t; /* PPU_NTSC or PPU_PAL */
typedef enum   ppu_state_t; /* Rendering state */
```

### PPU Registers (memory-mapped)

| Constant | Address | Description |
|---|---|---|
| `REG_PPUCTRL` | `$2000` | PPU control register |
| `REG_PPUMASK` | `$2001` | PPU mask register |
| `REG_PPUSTATUS` | `$2002` | PPU status register |
| `REG_OAMADDR` | `$2003` | OAM address |
| `REG_OAMDATA` | `$2004` | OAM data |
| `REG_PPUSCROLL` | `$2005` | Scroll position |
| `REG_PPUADDR` | `$2006` | VRAM address |
| `REG_PPUDATA` | `$2007` | VRAM data |
| `REG_OAMDMA` | `$4014` | OAM DMA (triggers CPU stall) |

### Functions

---

#### `ppu_init` / `ppu_destroy`

```c
int ppu_init(void);
int ppu_destroy(void);
```

Initialize and finalize PPU internal structures.

**Returns:** `0` on success, `-1` otherwise.

---

#### `ppu_reset`

```c
void ppu_reset(void);
```

Reset the PPU to its initial state (clears registers, scanline counter, shift
registers, etc.).

---

#### `ppu_set_cartridge`

```c
int ppu_set_cartridge(cartridge_t *cartridge);
```

Attach a cartridge to the PPU (provides access to CHR ROM / CHR RAM and the
nametable mirroring mode).

| Parameter | Direction | Description |
|---|---|---|
| `cartridge` | in | Loaded cartridge struct. |

**Returns:** `0` on success, `-1` otherwise.

---

#### `ppu_set_video`

```c
void ppu_set_video(ppu_video_t video);
```

Select the video standard.

| Parameter | Values |
|---|---|
| `video` | `PPU_NTSC` or `PPU_PAL` |

> **Note:** This function does not reset the PPU. When switching
> between video modes, call `ppu_reset()` after `ppu_set_video()`.

> Only NTSC is currently implemented in the PPU.

---

#### `ppu_get_video`

```c
ppu_video_t ppu_get_video(void);
```

Return the current video standard.

---

#### `ppu_run` / `ppu_stop`

```c
int ppu_run(void);
int ppu_stop(void);
```

Enable or disable the PPU rendering process.

- `ppu_run`: **Returns** `0` on success, `-1` if rendering is already enabled.
- `ppu_stop`: **Returns** `0` on success, `-1` if rendering is already stopped.

---

#### `ppu_clock`

```c
void ppu_clock(void);
```

Perform one clock tick in the PPU rendering pipeline. Called internally by
`sbus_clock()`. Advances the scanline/cycle counters, fetches background and
sprite data, calls `gui_set_pixel()` for each visible pixel, and triggers
NMI at VBlank via `cpu_trigger_nmi()`.

---

#### `ppu_reg_read` / `ppu_reg_write`

```c
uint8_t ppu_reg_read(uint16_t reg);
void    ppu_reg_write(uint16_t reg, uint8_t value);
```

Read or write a PPU memory-mapped register. Called by `sbus_read` / `sbus_write`
when the address falls in the PPU register range (`$2000`–`$3FFF`).

---

#### `ppu_set_mapper_trigger_cb`

```c
int ppu_set_mapper_trigger_cb(mapper_t *m, void (*trigger_cb)(mapper_t *));
```

Register a mapper scanline-counter callback. Required by mappers that count PPU
scanlines (notably MMC3 / Mapper 4) to generate IRQs.

| Parameter | Direction | Description |
|---|---|---|
| `m` | in | Mapper struct. |
| `trigger_cb` | in | Callback invoked when the PPU finishes the visible portion of a scanline (rendering must be enabled). |

**Returns:** `0` on success, `-1` otherwise.

## APU — Audio Processing Unit

**Header:** `src/include/apu.h`  
**Implementation:** `src/apu.c`

The APU synthesizes NES audio (two pulse/PWM channels, one triangle, one noise,
and one DMC) and writes PCM samples into the ring buffer.

### APU Registers (memory-mapped)

| Constant | Address | Description |
|---|---|---|
| `REG_PWM0`–`REG_PWM0_RAWH` | `$4000`–`$4003` | Pulse channel 1 |
| `REG_PWM1`–`REG_PWM1_RAWH` | `$4004`–`$4007` | Pulse channel 2 |
| `REG_TRI_CTR`–`REG_TRI_TMRH` | `$4008`–`$400B` | Triangle channel |
| `REG_NOISE_VOL`–`REG_NOISE_LC` | `$400C`–`$400F` | Noise channel |
| `REG_DMC`–`REG_DMC_SLEN` | `$4010`–`$4013` | DMC channel |
| `REG_STATUS` | `$4015` | APU status |
| `REG_FRM_SEQ` | `$4017` | Frame sequencer |

### Functions

---

#### `apu_init` / `apu_destroy`

```c
int apu_init(void);
int apu_destroy(void);
```

Initialize and finalize APU internal structures. `apu_init()` also calls
`audio_set_cb(apu_get_samples, ...)` to register the fill callback with the audio
device.

**Returns:** `0` on success, `-1` otherwise.

---

#### `apu_reset`

```c
void apu_reset(void);
```

Reset the APU to its initial state (silences all channels, resets frame sequencer).

---

#### `apu_clock`

```c
void apu_clock(void);
```

Perform one clock tick in the APU. Called internally by `sbus_clock()`. Advances
the frame sequencer, updates channel timers, and pushes a new PCM sample to the
ring buffer when appropriate.

---

#### `apu_reg_read` / `apu_reg_write`

```c
uint8_t apu_reg_read(uint16_t reg);
void    apu_reg_write(uint16_t reg, uint8_t value);
```

Read or write an APU memory-mapped register. Called by `sbus_read` / `sbus_write`
when the address falls in the APU register range.

---

#### `apu_get_samples`

```c
void apu_get_samples(void *data, uint8_t *stream, int len);
```

**APU audio fill callback.** Retrieves PCM samples from the ring buffer and writes
them into the audio output stream. This function is registered with the HAL via
`audio_set_cb()` and is called directly by the platform audio driver at 44100 Hz.

| Parameter | Direction | Description |
|---|---|---|
| `data` | in | Opaque user data (the APU's own state pointer). |
| `stream` | out | Audio output buffer to fill. |
| `len` | in | Number of bytes to fill. |

## Cartridge

**Header:** `src/include/cartridge.h`  
**Implementation:** `src/cartridge.c`

Loads a ROM file, decodes the iNES / NES 2.0 header, allocates PRG/CHR memory,
and initializes the appropriate mapper.

### Types

```c
typedef struct cartridge_t; /* Game cartridge */
typedef enum   mem_t;       /* Memory address space: CPU bus or PPU bus */
```

### Functions

---

#### `cartridge_load`

```c
int cartridge_load(const char *pathname, cartridge_t **cartridge);
```

Load a ROM file and prepare a cartridge.

| Parameter | Direction | Description |
|---|---|---|
| `pathname` | in | Path to the ROM file. |
| `cartridge` | out | Allocated and initialized cartridge struct. |

**Returns:** `0` on success; error codes include:
- `-ENOTSUP` — mapper not implemented.
- `-ENOMEM` — memory allocation failure.
- `-EINVAL` — invalid ROM format.

---

#### `cartridge_unload`

```c
void cartridge_unload(cartridge_t *cartridge);
```

Finalize and free a previously loaded cartridge (PRG ROM, CHR ROM/RAM, mapper state).

---

#### `cartridge_info`

```c
const cartridge_t *cartridge_info(cartridge_t *cartridge);
```

Return the cartridge information.

---

#### `cartridge_mem_read` / `cartridge_mem_write`

```c
uint8_t cartridge_mem_read(cartridge_t *cartridge, uint16_t address, mem_t memspace);
void    cartridge_mem_write(cartridge_t *cartridge, uint16_t address,
                             uint8_t value, mem_t memspace);
```

Read or write cartridge memory, routed through the active mapper.

| Parameter | Direction | Description |
|---|---|---|
| `cartridge` | in | Cartridge struct. |
| `address` | in | Memory address. |
| `memspace` | in | `CPU_SPACE` (PRG ROM/RAM) or `PPU_SPACE` (CHR ROM/RAM). |
| `value` | in | Value to write (`cartridge_mem_write` only). |

|---|---|---|
| `REG_JOYPAD1` | `$4016` | Joypad 1 register |
| `REG_JOYPAD2` | `$4017` | Joypad 2 register |

### Functions

---

#### `controller_reset`

```c
void controller_reset(void);
```

Reset joypad shift registers to their initial state.

---

#### `controller_reg_read` / `controller_reg_write`

```c
uint8_t controller_reg_read(uint16_t reg);
```

## Ring Buffer

**Header:** `src/include/ringbuffer.h`  
**Implementation:** `src/ringbuffer.c`

A ring buffer of `int16_t` samples used to pass PCM audio from the APU
(emulator thread) to the audio callback (audio thread). This is the core
synchronization primitive that governs emulation timing.

### Types

```c
typedef struct ringbuffer_t;  /* Ring buffer struct */
```

### Functions

---

#### `rbuff_init`

```c
int rbuff_init(ringbuffer_t *buffer, unsigned long capacity);
```

Initialize a new ring buffer.

| Parameter | Direction | Description |
|---|---|---|
| `buffer` | in/out | Ring buffer struct to initialize. |
| `capacity` | in | Maximum number of `int16_t` elements (set to `AUDIO_BUFFER_LEN`). |

**Returns:** `0` on success, error code otherwise.

---

#### `rbuff_destroy`

```c
void rbuff_destroy(ringbuffer_t *buffer);
```

Destroy a ring buffer and release its resources.

---

#### `rbuff_put`

```c
int rbuff_put(ringbuffer_t *buffer, int16_t element);
```

Push a new PCM sample into the ring buffer. **Blocks** when the buffer is full,
providing the backpressure that paces the emulator thread to real time.

| Parameter | Direction | Description |
|---|---|---|
| `buffer` | in/out | Ring buffer. |
| `element` | in | PCM sample (`int16_t`). |

**Returns:** `0` on success, error code otherwise.

---

#### `rbuff_get`

```c
int16_t rbuff_get(ringbuffer_t *buffer);
```

Remove and return one PCM sample from the ring buffer. Called from the audio
callback (`apu_get_samples`) to drain samples for playback. **Blocks** when the
buffer is empty.

| Parameter | Direction | Description |
|---|---|---|
| `buffer` | in/out | Ring buffer. |

**Returns:** The `int16_t` PCM sample.

## Mapper Interface

**Header:** `src/include/mapper.h`  
**Implementation:** `src/mappers/m0_NROM.c` … `m7_AxROM.c`, `src/mappers/mappers.c`

Mappers extend the NES address space beyond the base 32 KB PRG and 8 KB CHR by
bank-switching ROM regions. rNES implements mappers 0, 1, 2, 3, 4, and 7.

### Types

```c
typedef struct mapper_t;  /* Mapper state and function pointers */
```

### Constants

| Constant | Value | Description |
|---|---|---|
| `NUM_MAPPERS` | — | Total number of registered mappers. |

### Registration Macro

```c
REGISTER_MAPPER(id, init_fn, reset_fn, finalize_fn, cpu_handler, ppu_handler)
```

Used in each mapper source file to self-register into the `mappers[]` table.

### Generic Mapper Functions

```c
int mapper_init(struct _mapper_t *m, cartridge_t *c);
int mapper_reset(struct _mapper_t *m);
int mapper_finalize(struct _mapper_t *m);
```

These dispatch to the mapper-specific implementation selected during
`cartridge_load()`.

### Shared VRAM Handlers (Mapper 0 baseline)

```c
uint8_t m0_vram_mem_handler(struct _mapper_t *m, enum mem_op op,
                             uint16_t address, uint8_t value);
uint8_t m0_palette_mem_handler(struct _mapper_t *m, enum mem_op op,
                                uint16_t address, uint8_t value);
```

Default VRAM and palette memory handlers shared by mappers that do not override
them. Parameters:

| Parameter | Description |
|---|---|
| `m` | Mapper struct. |
| `op` | `READ` or `WRITE`. |
| `address` | Memory address. |
| `value` | Value to write (when `op` is `WRITE`). |

**Returns:** Value read/written from/to memory.
